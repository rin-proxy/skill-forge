# 🔨 skill-forge

> 🔒 **Diaudit & aman** — lolos skill-gate 100/100 · berjalan **lokal-saja**: tanpa panggilan jaringan, tanpa `eval`/kode ter-obfuscate. Semua kode terbuka & bisa kamu verifikasi sendiri. _(Audit keamanan: 2026-07-17)_

Scaffold, quality-gate, and ship installable OpenClaw skills. Bundles a /100 quality gate plus scaffold/README/CHANGELOG generators so a new skill goes from empty folder to a gate-passing, drop-in package. Use this when authoring a new OpenClaw skill or before publishing one.

**Version:** 1.0.0 · **Triggers:** create a skill, scaffold a skill, gate my skill, score this skill, author an openclaw skill, is my skill ready to ship

## Usage
Full guide and procedure: see **SKILL.md**. Bundled scripts (if any) live in **scripts/**; deep reference material in **references/** (loaded on demand).

## Install
    openclaw skills install git:rin-proxy/skill-forge

Or copy this folder into your OpenClaw workspace/skills/ directory.
